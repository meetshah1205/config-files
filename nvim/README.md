## How to use
1. Install [`vim-plug`](https://github.com/junegunn/vim-plug) .
2. Go to your `init.vim` file.
3. Paste the code
4. Run `:PlugInstall`
5. exit
6. Run `npm install -g eslint prettier`
7. Open neovim
8. Run: `:CocInstall coc-tsserver coc-eslint coc-prettier`
9. exit
10. open a file and enjoy

## Prerequisites
1. Neovim
2. NodeJS
